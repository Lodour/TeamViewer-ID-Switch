# TeamViewer-ID-Switch
Switch TeamViewer-ID to bypass commercial use detection on Mac OS.

## Usage
```sh
sudo python switch.py
```

## Note
* For TeamViewer 14+, `codesign` is needed to fix the `Code Signature Valid` error.
